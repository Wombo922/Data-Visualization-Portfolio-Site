Tech Stock Correlation Analysis

Analyzed 2 years of FAANG stock performance to uncover correlation patterns and identify which tech giants move together. This analysis reveals surprising independence in some stocks while others show strong synchronized movements.

üìä Explore Interactive Chart

Key Findings:

  • MSFT & META show strongest correlation (0.62) - These stocks move together more than any other pair
  • NVDA most independent - Lower correlations suggest unique market drivers
  • All stocks positively correlated (0.37-0.62) - Tech sector moves as a cohesive unit
  • Diversification insights: Holding multiple FAANG stocks provides less risk reduction than expected
Methodology & Data Sources

Stocks Analyzed: AAPL, MSFT, GOOGL, META, NVDA

Tools: Python, pandas, matplotlib, Polygon.io API

Cryptocurrency Volatility & Risk Analysis

Comprehensive risk-return analysis of major cryptocurrencies over 365 days. This project quantifies the volatility differences between established coins like Bitcoin and newer altcoins, helping investors understand the true risk-reward tradeoffs.

üìä Explore Interactive Risk Map

Key Findings:

  • ETH delivered 88.6% annualized returns - Highest performer despite high volatility (74.6%)
  • SOL shows 81.6% returns with 83.9% volatility - High risk, high reward profile
  • BTC offers 76.7% returns at "only" 42.8% volatility - Best risk-adjusted performance
  • MATIC underperformed with -8.0% returns - Demonstrates crypto downside risk
  • Higher volatility doesn't guarantee higher returns - Risk management crucial
Methodology & Data Sources

Data Source: CoinGecko API
Time Period: 365 days
Cryptocurrencies: BTC, ETH, SOL, MATIC
Metrics Calculated: 30-day rolling volatility, annualized returns, Sharpe ratios, risk-return profiles
Tools: Python, pandas, matplotlib, seaborn, CoinGecko API

DeFi Protocol Market Intelligence

Tracked Total Value Locked (TVL) trends across major DeFi protocols over 6 months to understand market share dynamics and protocol growth rates. Reveals which protocols are gaining traction and which are losing ground in the competitive DeFi landscape.

üìä Explore Interactive TVL Dashboard

Key Findings:
